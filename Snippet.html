Snippet explanation

The snippet we are going to examine is below.

// Want to search the web? Try Google //

Underneath the hood, this looks like: 

Want to search the web? Try <a href= "https://google.com" id= "the-best-link" class= "banana"  target= "blank"> Google </a>

Tags define how the browser formats and displays the page. They are denoted by <> to open and to close.

The tags present here are...

Paragraph Tag. Made by 
( <p> &  </p>)
-Formatts the snippet tag into a seperate line.
-Learn more here and here
Hyperlink tag. Made by 
( <a> &  </a>)
-Links to a seperate document
-Learn more here and here

Nestling is when one tag is put underneath the umbrella of another one. This is oftentimes compared to parent-child relationships.

So in our original snippet, the hyperlink tag is nestled within the paragraph tag which is also nestled under the overarching HTML tags

Our snippet also contains attribute names within the Hyperlink tag. Attributes are used to refer to HTML elements so they can be modified using JS or CSS

Our snippet has attributed values of...

ID
-Creates a unique identifier for this element
-Value of "the best link"
Class
-Creats a class also to identify but does not have to be unique to this element
-Value of "bannana"
Target
-Specifies where to open the linked document
-Value of "blank" so it opens it in a new window
