Command Line

The command line is how you interact with the computer through basic steps communicated to the OS. On Windows, the Git Bash application provides a terminal for the user to interact with the OS. On Mac, you can access the terminal by performing Command + Space.

Commands

  • cd to navigate between directories
  • ls to list the contents of a directory
  • mkdir to create an empty directory
  • touch to create an empty file
  • code . to open VS Code

Publishing HTML

Surge allows you to easily deploy static sites directly to the Web. After you input the download commands from Surge.sh on your terminal and create an account, follow these steps to publish your site

  1. Cd to your project
  2. Deploy by typing surge

You can edit and redeploy to the same domain by following these steps

  • surge --domain (domain name)
  • echo (domain name) > CNAME

A HTML validator site will check the syntx of your HTML after you've publishing
